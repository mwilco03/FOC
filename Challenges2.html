  • ConvertFrom-Json / ConvertTo-Json
    • Parse API responses or structure custom output
    • Used in scripts to serialize/deserialize config and logs
  • ConvertFrom-Csv / Export-Csv
    • Import/export data tables from files or APIs
    • Used for reporting, logging, and automation pipelines
  • ConvertFrom-StringData
    • Converts .ini-style config blocks to hash tables
  • Invoke-WebRequest / Invoke-RestMethod
    • Used to pull files, fetch APIs, or query REST services
    • Frequently combined with JSON and CSV converters
  • New-Object System.Net.WebClient
    • Legacy file downloading method (use Invoke-WebRequest for modern)
  • Start-Process / Wait-Process
    • Run programs and wait for them to complete
    • Automate GUI tools, e.g., wait for Notepad to close
  • Out-GridView -PassThru
    • Used for user-interactive filtering of pipeline objects
  • Group-Object
    • Used for frequency analysis and deduplication
  • Select-Object with calculated properties
    • Create on-the-fly custom columns like memory in MB or file age
  • $PSBoundParameters
    • Check what params were actually supplied to a function
  • Read-Host -AsSecureString + [Runtime.InteropServices.Marshal]
    • Used to collect passwords securely and (optionally) convert for testing
  • New-Variable / dynamic variable access
    • Create and access variable names programmatically
