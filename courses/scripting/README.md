# Scripting Challenge Labs

Each subdirectory is a language-specific challenge lab with interactive reference, drills, and quizzes.

## Labs

| Lab | Directory | Drills | Quiz |
|-----|-----------|--------|------|
| Python | `python/` | 20 challenges | 20 questions |
| PowerShell | `powershell/` | 20 challenges | 20 questions |
| Bash | `bash/` | 20 challenges | 20 questions |
| Batch | `batch/` | 15 challenges | 15 questions |
| Data & APIs | `data-apis/` | 20 challenges | 20 questions |

Each `.html` file is self-contained. The companion source files (`.py`, `.ps1`, `.sh`, `.cmd`) contain the raw challenge scripts.
