# Networking Fundamentals

Interactive networking study tools. Zero dependencies — open any `.html` in a browser.

## Tools

| Tool | File | Description |
|------|------|-------------|
| **NetDrill** | subnetting.html | 7-tab study platform (OSI, protocols, number systems, subnet math, calculator, practice, reference) |
| **Subnet Invaders** | invaders.html | Space Invaders quiz game — 8 waves |
| **Subnet Quest** | quest.html | Turn-based RPG with boss-fight subnet tables |
| **OSI Tower** | tower.html | Progressive 3-stage OSI model builder |
| **Bit Flip** | bitflip.html | Toggle 32 bits to build CIDR subnet masks |
| **Hex Trainer** | hex.html | Guided long-division hex conversion |
| **PCAP Forensics** | pcap-challenge.html | Simulated packet capture analysis |
| **Syntax Drill** | syntax-drill.html | Command syntax notation practice |
| **Subnetting Sheet** | SubnettingSheet.html | Interactive editable reference |
