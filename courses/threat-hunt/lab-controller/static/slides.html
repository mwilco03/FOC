The Tuning Problem

  • A public kiosk generates ~25,000 Sysmon events per hour
  • 99% is noise: browser updates, Defender, svchost, telemetry
  • The attack is 127 events buried in 47,000

The Workflow

  1. Week 1: Collect everything. Drown in noise.
  2. Week 2: Exclude top noise sources. 50,000 → 500.
  3. Week 3: What remains is worth alerting on.

Today you compress 3 weeks into 30 minutes.

Zeek: Network Ground Truth

  • conn.log — every connection (the backbone)
  • dns.log — every DNS query
  • http.log — HTTP transactions
  • ssl.log — TLS handshakes, JA3
  • smb.log — file share access

What to Look For

  • Regular-interval DNS queries (C2 beacon)
  • Long subdomain labels (DNS tunneling)
  • SMB from public → internal VLAN (lateral movement)
  • High bytes_out from single host (exfil)

Containment: When You Find It

# Default-block architecture (NOT allow-beats-block)
Set-NetFirewallProfile -All -DefaultInboundAction Block -DefaultOutboundAction Block

# Then punch holes for:
#   Jump box (your access)
#   ELK (keep logs flowing)
#   DNS (to your DC only)
#   Loopback

# Verify: can you reach ELK? YES. Can you reach internet? NO.
Block wins over Allow in Windows Firewall. Use default-block + explicit allows. Never rely on allow beating block.

Kansa: Remote Forensic Collection

  • PowerShell scripts over WinRM — no agent needed
  • Run from your terminal against the victim
Get-ScheduledTask | Where { $_.Actions.Execute -like '*powershell*' }
Get-ItemProperty HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
Get-CimInstance -Namespace root/subscription -ClassName __EventConsumer
Get-WinEvent -FilterHashtable @{LogName='Security';ID=4698}

Every one of these commands finds a different piece of this attack.

Playbook PB-01: Compromised Terminal

  1. Isolate — Pull network at switch port. Do NOT touch the machine.
  2. Preserve — Do NOT reboot (Deep Freeze will destroy evidence).
  3. Contain — Run Invoke-Contain.ps1 via PS Remoting.
  4. Collect — Kansa triage collection, Winlogbeat still shipping.
  5. Analyze — Kibana + Arkime. Build timeline.
  6. Remediate — Reimage from golden. Update AppLocker + DNS blocklist.
  7. Report — Brief leadership. Document IOCs. Update playbook.
