# Courses

Each subdirectory is a self-contained course. No Docker required — these are pure static HTML/JS.

## Course Contract

A course directory **must**:
1. Be usable with no dependency on sibling course directories
2. Include a `README.md` explaining what it covers
3. Work by opening `.html` files directly in a browser (no build step)
