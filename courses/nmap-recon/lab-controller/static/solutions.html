🔓 Instructor Solution Guide

Host Discovery (200 pts)

Ping Sweep (50 pts)
FLAG{13}
nmap -sn 172.20.1.0/24

Count target hosts only (exclude gateway .1 and student IPs .101-.110)

Find the Mail Server (50 pts)
FLAG{172.20.1.30}
nmap -p 25 172.20.1.0/24
Ghost Host (100 pts)
FLAG{172.20.1.50}
nmap -Pn -p 2222 172.20.1.50
Teach: ICMP is blocked on .50, but ARP discovery on local subnet still finds it. The real lesson is -Pn for cross-subnet scanning.

Port Scanning (550 pts)

Non-Standard Web Port (100 pts)
FLAG{apache_on_8888}
nmap -p- 172.20.1.11    # finds 8888
nmap -sV -p 8888 172.20.1.11
Database Count (100 pts)
FLAG{4}
nmap -p 3306,5432,6379,27017 172.20.1.0/24

MySQL (.20), Postgres (.21), Redis (.22), MongoDB (.23)

Custom App Ports (150 pts)
FLAG{4}
nmap -p- 172.20.1.70    # 4444, 5555, 6666, 9999
Full Port Scan (200 pts)
FLAG{2}
nmap -p- 172.20.1.80    # 31337 and 65000
Teach: 65000 is slow to respond. Students may need --host-timeout

Service Detection (600 pts)

MySQL Version (150 pts)
FLAG{8.0}
nmap -sV -p 3306 172.20.1.20
SSH Version (150 pts)
FLAG{ssh_version_detected}
nmap -sV -p 2222 172.20.1.50
FTP Banner Grab (150 pts)

nmap -sV -p 21 172.20.1.40
# OR: nc 172.20.1.40 21
SMTP Banner (150 pts)
FLAG{smtp_banner_captured}
nmap -sV -p 25 172.20.1.30

Advanced Scanning (1450 pts)

Elite Port (200 pts)
FLAG{elite_port_31337}
nc 172.20.1.80 31337
Teach: 31337 = "elite" in leet speak. Classic backdoor port.
UDP Discovery (250 pts)
FLAG{udp_service_discovered}
nmap -sU -p 4444 172.20.1.80
Note: UDP may not work on Docker Desktop Windows. Give partial credit.
Slow Service (250 pts)
FLAG{patience_rewarded_65000}
nc 172.20.1.80 65000    # wait 3 seconds
OS Detection (200 pts)
FLAG{Linux}
sudo nmap -O 172.20.1.10
Outside the Top 1000 (300 pts)
FLAG{port2_not_in_top1000}
nmap 172.20.1.90         # nothing!
nmap -p- 172.20.1.90     # finds port 2
nmap -sV -p 2 172.20.1.90
Teach: This is the biggest aha moment. Default nmap misses it. Always use -p-.
Hidden in Headers (250 pts)
FLAG{http_header_hidden}
curl -I http://172.20.1.10    # X-Secret-Flag header

NSE Scripts (1200 pts)

Anonymous FTP (200 pts)
FLAG{ftp_anonymous_access}
nmap --script=ftp-anon -p 21 172.20.1.40
Redis Exposed (200 pts)
FLAG{redis_no_auth_dangerous}
nmap --script=redis-info -p 6379 172.20.1.22
Teach: Unauthenticated Redis = full server compromise. Real-world critical finding.
